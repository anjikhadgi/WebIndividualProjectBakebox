#  Interior Home Design Inspiration - Backend

This is the **backend** for the **Interior Home Design Inspiration Platform**, built using **Node.js**, **Express.js**, and **PostgreSQL**.
It provides RESTful APIs for managing design themes, categories, images, and user data.

---

##  Features

- REST API for fetching and managing interior design themes
- PostgreSQL database integration
- Express middleware for routing and error handling
- CORS setup for frontend communication
- Environment-based configuration using `.env`

---

##  Tech Stack

- Node.js
- Express.js
- PostgreSQL
- Sequelize 
- dotenv
- CORS

---
